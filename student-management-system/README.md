# Student Management System 🎓

A beginner-friendly Python project to manage student records using the terminal.

## Features
- Add a student
- View all students
- Search for a student
- Delete a student
- Save student records to a file
- Load student records from a file
- Handle invalid input safely using exception handling

## Concepts Used
- Variables and input
- Conditions and loops
- Functions
- Lists and dictionaries
- File handling
- Exception handling
- Modules/imports (if expanded later)

## Files
- `main.py` → main Python program
- `students.txt` → stores student data
- `README.md` → project documentation
